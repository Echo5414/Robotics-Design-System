## Assets

Assets are stored in S3 and synced on-demand. Run:

```bash
# Sync all assets
node scripts/sync-assets.js

# Check what would be downloaded
node scripts/sync-assets.js --check

# Force re-download all
node scripts/sync-assets.js --force
```

Or add to package.json:

```json
{
  "scripts": {
    "assets:sync": "node scripts/sync-assets.js",
    "assets:check": "node scripts/sync-assets.js --check"
  }
}
```
